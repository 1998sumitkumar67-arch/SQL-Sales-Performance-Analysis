SQL Sales Performance Analysis Using MySQL
Project Overview

This project demonstrates an end-to-end SQL-based business analytics workflow using the Superstore Sales Dataset. The objective of the project is to analyze business performance, identify profitable and loss-making areas, and generate actionable business insights using MySQL.
