# 📌 Project Overview

This project demonstrates an **end-to-end SQL-based Sales Performance Analysis** using the **Superstore Sales Dataset**. The objective is to analyze business performance, identify profitable and loss-making areas, uncover trends, and generate actionable business insights using **MySQL**.

This project showcases practical SQL skills, business analytics thinking, and data-driven decision-making capabilities required for a **Data Analyst role**.

---

# 🎯 Project Objectives

The main objectives of this project are:

- ✅ Perform data validation and cleaning
- ✅ Calculate important business KPIs
- ✅ Analyze product profitability
- ✅ Identify loss-making products
- ✅ Analyze category and sub-category performance
- ✅ Analyze regional and state performance
- ✅ Analyze city-level business performance
- ✅ Analyze customer segments
- ✅ Analyze shipping methods
- ✅ Perform year-wise trend analysis
- ✅ Generate actionable business recommendations
